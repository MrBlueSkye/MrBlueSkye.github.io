Overview

Routing Protocols are software algorithms used to determine the best path to any given IP address. The two defined classes of routing protocols are Distance Vector, and Link-State.

Distance Vector

Distance Vector protocols learn their immediate surrounding networks and determine their own best path to each network that it is aware of, and will create a Routing Table with each best path.. Once a best path to each network has been computed, the routers will then share their routing tables with oneanother, so that they all learn only the best routes. This is an effieciency of DV protocols, but also a drawback in that other routers are only made aware of the best path to a given network, and no alternative routes.

Link State

Link State protocols do not focus on the "best" path, but instead they learn ALL the possible routes to any network they are aware of. They then create a database entry in for each network in a Link-State Database (LSDB). By maintaining a complete map of the network, they can make informed decisions at each hop to choose the optimal path, and also maintain a database of possible backup paths should their preffered path be impacted.
